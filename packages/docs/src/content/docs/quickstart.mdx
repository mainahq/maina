<Steps>

1. **Install**

   ```bash
   curl -fsSL https://api.mainahq.com/install | bash
   ```

2. **Initialize your project**

   ```bash
   maina init --install
   ```

3. **Commit with verification**

   ```bash
   maina commit
   ```

</Steps>

That's it. Maina detects your stack, installs verification tools, and runs the full pipeline on every commit.
